# FinSpace 基金管理功能使用说明

## 功能介绍
新增基金管理功能，可以自动获取基金最新净值，自动计算持仓收益。

## 使用步骤

### 1. 安装Python依赖
```bash
cd backend
pip install -r requirements.txt
```

### 2. 启动后端服务
```bash
python main.py
```
服务会运行在 http://localhost:8000，确保启动成功再进行下一步。

### 3. 记录基金买入
1. 进入「投资交易」页面
2. 点击「记交易」
3. 选择「买入」类型
4. 标的代码填写6位数字基金代码（比如：000001）
5. 填写买入数量、单价等信息
6. 保存记录

### 4. 更新基金净值
1. 切换到「持仓分析」标签
2. 点击右上角「更新基金净值」按钮
3. 系统会自动调用后端接口，获取所有持仓基金的最新净值
4. 自动更新持仓收益、收益率、总市值等数据

## 注意事项
- 基金代码必须是6位数字，比如：000001（华夏成长混合）、161725（招商中证白酒指数）
- 后端服务需要保持运行才能更新净值数据
- 数据来源于东方财富网，更新频率每天一次（收盘后更新）
- 仅支持国内公募基金的净值查询

## 常见问题
Q: 点击更新基金净值提示连接失败？
A: 请检查backend服务是否正常启动，端口8000是否被占用。

Q: 部分基金更新失败？
A: 请检查基金代码是否正确，确保是6位数字的国内公募基金代码。

Q: 每次使用都要启动后端吗？
A: 只有在需要更新净值的时候才需要启动后端，日常查看数据不需要。
